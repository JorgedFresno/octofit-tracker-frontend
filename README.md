# OctoFit Tracker - Frontend

React 19 + Vite frontend for the OctoFit Tracker application.

## Development Server

Runs on port **5173**

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```

## Features

- ⚛️ React 19
- ⚡ Vite for fast development
- 🎯 TypeScript support
- 🔗 API proxy to backend (localhost:8000)
- 📱 Responsive design ready

## Port Configuration

- **Frontend Dev Server**: 5173
- **Backend API**: 8000 (proxied via `/api`)
